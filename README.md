“货比百家”的美妆商品导购系统
项目简介:
在这个“爱美”的时代，美妆品逐渐成为人们生活中必不可少的物品。如今，不仅
女性化妆，男性也越来越注重个人形象。然而，如何才能买到物美价廉、且又适合
自己皮肤的商品是一个令人头疼问题。“‘货比百家’的美妆商品导购系统”可以
为用户跨网站规模的搜索网上美妆商品,并根据商品的特征和用户的个人信息为用
户提供合适的、可靠的商品。
使用场景:
该系统的使用场景有：
（1）当一个用户想要网购一件在指定的价格范围内、某个知名品牌的美妆商品，
用户只用在系统的用户界面上输入这些商品信息，就可以看到来自各大电商网站
（唯品会、天猫等等）和品牌官网或者旗舰店的美妆商品们。
（2）当一个用户购买适合自己使用的商品，但却没有明确的商品特征时，她/他可
以在系统的用户界面输入自己的个人信息，系统根据用户的个人信息，为用户推荐
可靠的商品。比如用户是干性皮肤，系统便为用户推荐适合干性皮肤的护肤品。
